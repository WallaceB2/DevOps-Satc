# 🔧 DevOps

# DevOps é uma cultura que une Desenvolvimento (Dev) e Operações (Ops).

Antes, quem criava o sistema e quem mantinha ele funcionando trabalhavam separados.
Isso causava muitos erros e atrasos.

O DevOps faz esses times trabalharem juntos usando automação, para:

- 🚀 Entregar atualizações mais rápido
- 🛠️ Reduzir erros
- 🔄 Automatizar testes e deploy
- 📦 Melhorar a organização do processo
- 🎬 Exemplo com a Netflix

Na Netflix, o DevOps ajuda a lançar:
- Novas funcionalidades
- Melhorias no aplicativo
- Correções de bugs
- Tudo isso de forma rápida e organizada, sem quebrar o sistema.

👉 Resumindo: DevOps é a forma moderna de desenvolver e entregar software.

🛡️ SRE (Site Reliability Engineering)

# SRE é uma área criada pela Google.
O foco do SRE é garantir que o sistema fique no ar e funcione bem.

Eles trabalham com:
- 📊 Monitoramento
- ⚙️ Automação
- 🚨 Resposta a incidentes
- 📈 Alta disponibilidade

# 🎬 Exemplo com a Netflix

Quando milhões de pessoas entram ao mesmo tempo para assistir uma nova temporada, o SRE garante que:

- O sistema aguente o pico de acesso
- O vídeo não trave
- O serviço não saia do ar

👉 Resumindo: SRE é o especialista em manter o sistema estável e confiável.
